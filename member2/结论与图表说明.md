# 地域与交易额分析（结论与图表说明）

## 数据与口径

- 数据来源：`as2_data/cleaned/ga_session.csv`、`ga_total.csv`、`ga_geo_network.csv`
- Join 粒度：以 `uid` 为键进行 session 级别拼接
- 指标：`total_transaction_revenue` 换算为美元：`revenue_usd = total_transaction_revenue / 1e6`
- 分析范围：以清洗后的数据为准，具体日期范围见 `outputs/run_metadata.txt`

## 可直接写入报告的结论（模板）

> 以下结论需结合实际生成的 TopN 排名与占比数值（可在 `outputs/` 的 CSV 中查到）进行微调。

1. **交易额在地域上呈现明显集中趋势**：少数国家贡献了大部分交易额，Top10 国家累积占比显著高于长尾国家。
2. **美国市场通常占据主导地位**（以本数据集为例）：从国家维度的聚合结果可观察到美国交易额远高于其他国家，说明当前电商交易主要由美国用户/订单贡献。
3. **大洲维度上北美洲/美洲贡献突出**：与国家结论一致，按大洲聚合时，美洲通常贡献最高交易额；其他大洲贡献相对分散。
4. **对运营/投放的启示**：
   - 若目标是“提升总体 GMV/交易额”，可优先在交易额贡献最高的国家/大洲加大投放、优化结算与物流体验。
   - 对交易额贡献较低但存在访问量的国家，可进一步分析流量质量（转化率、客单价）决定是否继续投入。
